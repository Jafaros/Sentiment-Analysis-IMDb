# Sentiment-Analysis-IMDb

Pro správné spuštění je potřeba složka aclImdb s recenzemi v root složce projektu a dále nainstalované balíčky v projektu:

1. `pip install nltk`

2. `pip install scikit-learn`

3. `pip install matplotlib`

Projekt lze následně spustit pomocí příkazu v root složce `python start.py`
